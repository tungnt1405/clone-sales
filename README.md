## Step Intall Application
- Step 1: pull code from branch master
- Step 2: open folder source
- Step 3: run command 
``` composer install ```
to install vendor
- Step 4: run 
``` cp .env.example .env ```
- Step 5: `npm install`
- Step 6: `yarn install`

## Run Application
- setting file .env(info database)
- run `npm run dev` || `yarn dev`
- run `npm run watch` || `yarn watch`
- run `php artisan serve`
